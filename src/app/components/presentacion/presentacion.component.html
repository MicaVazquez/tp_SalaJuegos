EXPLICACION DEL JUEGO

Memoria Maestra

Verás varias cartas boca abajo. Cada carta tiene una imagen única en su parte posterior. Toca una carta para voltearla y revelar la imagen. Luego, toca otra carta para intentar encontrar la misma imagen que la primera. Si las dos cartas que eliges coinciden, ¡habrás encontrado una pareja y se quedarán boca arriba! Si las cartas no coinciden, se voltearán nuevamente boca abajo y tendrás que recordar su ubicación para futuras jugadas. Repite este proceso antes que termine el tiempo y GANA!
